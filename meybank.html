Meybank Online Banking

Project information

  • Category: Web design
  • Project date: 20 February, 2023
  • Language: Java Web MVC / SSMS
  • Project URL: No Hosting

This project is a school assignment, simulating an online banking website modeled after Public Bank. It includes features like transfers, payments, loans, printing, and OTP. The site is user-friendly and written in Java. Users can easily transfer funds, apply for loans, and the system locks accounts after multiple failed login attempts to ensure security.
